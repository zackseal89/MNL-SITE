# MN LEGAL — Headless Web Application System

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fzackseal89%2FMNL-SITE&env=WORDPRESS_API_URL&project-name=mn-legal-web&repository-name=mnl-site)

This repository contains the complete frontend architecture and design documentation for the **MN Legal (MN Advocates LLP)** web ecosystem.

## 📁 Repository Overview
- `/mn-legal-web`: The core Next.js application. [Go to App README](./mn-legal-web/README.md)
- `MN-Legal-Webpage-Master-Prompt.md`: The single source of truth for the design system and brand tokens.
- `MN-Legal-WordPress-Integration.md`: Technical documentation for the headless WordPress/WPGraphQL integration.

## 🚀 Key Features
- **Cinematic Frontend:** High-end scroll animations with GSAP and custom CSS.
- **Headless CMS:** Powered by WordPress, consumed via GraphQL.
- **Performance Optimized:** Built with Next.js 15+ for blazing fast load times and SEO.
